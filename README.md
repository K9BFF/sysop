# sysop: system operations
a nixos operations frontent for lazy people

### the only prerequesite is `nom`. please make sure it's installed before using `sysop`!

### options:


| flag |        function        |
|------|------------------------|
|-r    | rebuild                |
|-u    | upgrade                |
|-d    | garbage collection     |
|-p    | nix-shell installation |
|-h    | help                   |
